Secure Chat Application with Image Steganography

This Angular-based application provides a secure platform for instant messaging, enhanced with image steganography to ensure message confidentiality.

Security and Encryption

Security is at the core of our chat application. We use state-of-the-art security mechanisms to ensure data confidentiality and integrity. The key features are:

  • End-to-End Encryption (E2EE): All messages are encrypted on the sender’s device and decrypted only on the receiver’s device, ensuring that no third party can read the messages.
  • AES Encryption: Advanced Encryption Standard (AES) is used for encrypting the messages. AES is one of the most secure encryption algorithms available, providing strong encryption with minimal performance overhead.
  • Data Privacy: User data is kept private and only shared with authorized parties. We comply with privacy laws and best practices to ensure that user information is handled securely and with consent.
  • Image Steganography: In addition to text-based messages, this application uses image steganography to hide secret messages within images. The steganography technique ensures that even if the encrypted image is intercepted, the hidden messages cannot be easily extracted without the correct decryption key.

Key Technologies

  • AES (Advanced Encryption Standard): A symmetric encryption algorithm used to encrypt user data. AES offers high security and efficiency, making it ideal for use in real-time communication systems like ours.
  • Firebase: Firebase is used as the backend for our application, providing a scalable real-time database. Firebase ensures fast and secure message delivery with authentication, real-time synchronization, and cloud storage services.

Features

  • Secure messaging using advanced image steganography.
  • Dynamic and user-friendly Angular-based interface.
  • Scalable modular code structure for enhanced development.

How to Run the Project

  1. Clone the repository: git clone https://github.com/prudhvinathreddy/chat-app.git
  2. Install Node.js and Angular CLI: npm install -g @angular/cli
  3. Open the project folder in Visual Studio Code.
  4. Run the command: ng serve
  5. Open http://localhost:4200 in your browser.
